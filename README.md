# 简介

web版本《明日方舟》角色Live 2D


# 本地开发

```bash

# 克隆本仓库
git clone https://github.com/SHUAXINDIARY/ArknightsLive2d-Web.git

# 进入目录
cd ArknightsLive2d-Web

# 启动
pnpm start

# 浏览器访问 http://localhost:3000/

```

# 数据来源

所有数据来自 [Ark-Models](https://github.com/isHarryh/Ark-Models)
